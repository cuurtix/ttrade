# ttrade

Mini jeu de stratégie en temps réel (prototype navigateur) inspiré de **0 A.D.**.

## Lancer le jeu

```bash
python3 -m http.server 8000
```

Puis ouvrez http://localhost:8000.

## Boucle de jeu

- Produisez des ressources (bois, nourriture, métal) automatiquement.
- Construisez des bâtiments pour accélérer l'économie.
- Formez des villageois et des guerriers.
- Lancez des raids pour gagner du prestige.
